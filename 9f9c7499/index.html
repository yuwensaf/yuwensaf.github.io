json 格式介紹

Posted on 2020-05-04 In JavaScript

🎃 JSON（JavaScript Object Notation，JavaScript 物件表示法，讀作/ˈdʒeɪsən/）

🎃 「json」是一種「資料交換的格式」，許多人都可以透過「json」來交換資料

🎃 會需要「json」的原因為：

每個程式語言（例如 PHP, Java）都會有自己的資料庫（例如 Microsoft SQL, MySQL）

當我需要「互相傳遞資料」時，因為資料庫的程式語言不同，就沒辦法做「資料的交換」。

因此，就需要有一個「共通的格式」來做資料交換，其中一種格式就是「json」

🎃 「json」的格式：其實就是透過「物件、陣列」來讀取資料
