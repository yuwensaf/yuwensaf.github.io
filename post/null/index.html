最終目的

我想要將這個網頁，上傳到 github pages，這樣就可以用網址開啟網頁了

網頁所包含的檔案，都放在資料夾「Dream School-jQuery」中如下圖紅框處

前置作業

  • 要在檔案資料夾，開啟終端機

在資料夾「Dream School-jQuery」按右鍵 > Services > New Terminal Tab at Folder

就會在資料夾「Dream School-jQuery」開啟終端機了

第一步

輸入指令 $ git init

  • 目的：要建立儲存庫

有成功建立儲存庫的話，資料夾裡會出現「.git」這個資料夾（git 資料夾預設是隱藏檔案，因此要把隱藏檔案顯示才能看到）

這時，輸入指令 $ git status，可以看到檔案都是用紅字顯示，是因為檔案都還沒 commit 到 git 裡面

第二步

輸入指令 $ git add .

  • 目的：將所有檔案 commit 到 git 裡面

$ git add . 最後面的那個「.」代表「所有檔案」

這時，再次輸入指令 $ git status，可以看到檔案都是用綠色字顯示，代表檔案都已經 commit 到 git 裡面了

第三步

輸入指令 $ git commit -m "first commit" .

  • 目的：將所有檔案 commit 到 master

$ git commit -m "first commit" . 最後面的那個「.」代表「所有檔案」

這時，再次輸入指令 $ git status，可以看到「On branch master
nothing to commit, working tree clean」

代表 commit 成功

第四步

回到 github 頁面上，將下圖中紅框處的指令，複製後貼到 terminal 中執行

$ git remote add origin git@github.com:yuwensaf/WorkShop-DreamSchool-jQuery.git

  • 目的：將遠端的檔案，加到此 repository 中

第五步

在同樣的 github 頁面上，將下圖中紅框處的指令，複製後貼到 terminal 中執行（檔案越多，此步驟會需要跑越久的時間）

$ git push -u origin master

  • 目的：將檔案 push 到此 repository 中

出現此訊息代表成功！

這時，回到 github 頁面上，點擊下圖中紅框處

就可以看到，所有的檔案都已經上傳到這個 repository 了

第六步

  • 進入 Setting

  • 往下滾到 GitHub Pages 的區塊
  • 將 Source 改成 master branch 後，就會看到這行字「Your site is published at https://yuwensaf.github.io/WorkShop-DreamSchool-jQuery/」
  • 用此網址，就可以進入我上傳的靜態網頁了（一開始打開網址可能會出現 404 的畫面，是因為 github pages 會需要一段時間才會將網頁準備好，耐心等一等就會出現了）

成功上傳靜態網頁了
