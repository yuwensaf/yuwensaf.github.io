for - i++ 寫法

Posted on 2020-05-04 In JavaScript

這裡要介紹「i++」的各種寫法

  • 設定一個變數 var i = 0;
  • console.log(i) 就會顯示「0」

要如何讓數字去做「+ 1」呢？

有三種寫法：

🍋 以下三種寫法，寫一次只會「+ 1」一次，要再「+ 1」就要再寫一次

🍋 下面是以「加法+」作為示範，減法的話就把「+」換成「-」即可

寫法一 i = i + 1

🎃 賦予i一個新的值，讓 i = i + 1;

  • 這樣，console.log(i) 就會顯示「1」

寫法二 i+=1

🎃 要讓 i = i + 1，也可以這樣寫：i+=1

🍋 因為現在的i已經是「1」了，所以i+=1就會等於「2」

寫法三 i++

🎃 要讓 i = i + 1，也可以這樣寫：i++

🍋 因為現在的i已經是「2」了，所以i++就會等於「3」
